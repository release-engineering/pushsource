Push items: cgw¶

class pushsource.CGWPushItem[source]¶

A PushItem representing a Content Gateway (CGW) YAML file.

For push items of this type, the src() attribute refers to a YAML file containing list of definitions for objects in Content Gateway.

This library does not verify that the referenced YAML file is valid.
